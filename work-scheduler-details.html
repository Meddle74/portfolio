  • Github Repository:

A simple calendar application where you can enter and save events. A color code helps you see when the events are to take place.Current date is displayed at the top of the sceduler and past events are colored grey. Current events are colored red and future events are colored green. Text can be entered by clicking the time block and you can save your text by clicking the save button. Events are saved in local storage.
